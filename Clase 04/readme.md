# Guía de Uso - Clase 04: Interfaces Gráficas con Tkinter

En esta clase se explora el desarrollo de aplicaciones con interfaces gráficas (GUI) utilizando la librería estándar de Python, `tkinter`.

## Contenido de la Carpeta

A continuación se detalla el funcionamiento del script incluido:

### Cajero Automático Profesional (`Banco.py`)
Un simulador de cajero automático con un diseño visual moderno, inspirado en terminales bancarias.
*   **Funcionamiento**: Utiliza `tkinter` para crear una interfaz gráfica con una paleta de colores personalizada (azul marino, dorado, teal).
*   **Características Principales**:
    - **Gestión de Cuentas**: Creación de perfiles de usuario con NIP.
    - **Operaciones**: Permite realizar depósitos y retiros de saldo.
    - **Historial**: Registra las transacciones en tiempo real.
    - **Diseño Moderno**: Botones estilizados con efectos hover y una pantalla LCD simulada.
*   **Cómo usar**: Al ejecutar el programa se abrirá una ventana de Windows con la interfaz del cajero. No es necesario realizar acciones por consola después de iniciar el script.

## Requisitos
Este programa requiere la librería `tkinter`, la cual usualmente viene incluida con Python. Si experimentas errores al ejecutarlo, asegúrate de tenerla instalada.

## Cómo Ejecutar el Código
1. Asegúrate de tener Python instalado.
2. Abre una terminal en esta carpeta.
3. Ejecuta el script usando el comando:
   ```bash
   python Banco.py
   ```
   *Se abrirá automáticamente la ventana de la aplicación.*
