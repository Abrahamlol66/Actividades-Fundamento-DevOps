# Guía de Uso - Clase 02: Manejo de Entradas y Estructuras de Control

En esta carpeta se explora el manejo interactivo de datos, validaciones y la creación de menús estructurados en Python.

## Contenido de la Carpeta

A continuación se detalla el funcionamiento del script incluido:

### Formulario Interactivo (`Formulario.py`)
Este programa solicita y gestiona datos personales mediante un flujo interactivo.
*   **Funcionamiento**: Realiza validaciones en tiempo real para cada dato ingresado (edad positiva, sexo M/F, nombre alfabético, estatura válida). Una vez recolectados, permite acceder a ellos a través de un menú.
*   **Funcionalidades del Menú**: 
    1. Mostrar edad.
    2. Mostrar sexo.
    3. Mostrar nombre.
    4. Mostrar estatura.
    5. Mostrar todos los datos recopilados.
    6. Salir.
*   **Cómo usar**: Ejecuta el archivo y completa los datos solicitados. Luego utiliza las opciones del menú para visualizar la información guardada.

## Cómo Ejecutar el Código
1. Asegúrate de tener Python instalado.
2. Abre una terminal en esta carpeta.
3. Ejecuta el script usando el comando:
   ```bash
   python Formulario.py
   ```
