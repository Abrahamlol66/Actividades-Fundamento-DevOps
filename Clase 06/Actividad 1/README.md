# Actividad 1 de la Clase 06: Análisis de Ventas con Múltiples Librerías

Esta actividad tiene como objetivo realizar un análisis de datos de ventas de coches, similar al de la Clase 05, pero utilizando más librerías de Python para reforzar el aprendizaje de estas herramientas.

## Librerías Utilizadas
- **pandas**: Para la manipulación y análisis de datos estructurados.
- **numpy**: Para operaciones numéricas y manejo de arrays.
- **matplotlib**: Para la creación de gráficos y visualizaciones.

## Descripción de la Actividad
El script `Actividad.py` realiza las siguientes tareas:
1. Crea un conjunto de datos de ventas de coches con información de vendedores, modelos, precios y comisiones.
2. Calcula la comisión en dinero por cada venta.
3. Agrupa los datos por vendedor para obtener estadísticas como número de coches vendidos, total de ingresos, promedio de venta y total de comisiones.
4. Ordena los resultados por ingresos totales.
5. Genera un gráfico de barras para visualizar los ingresos y comisiones por vendedor.

## Cómo Ejecutar
1. Asegúrate de tener Python instalado en tu sistema.
2. Instala las dependencias necesarias ejecutando `pip install pandas numpy matplotlib`.
3. Ejecuta el script con `python Actividad.py`.

## Notas
Esta actividad refuerza el uso de pandas para análisis de datos, incorporando numpy para cálculos numéricos y matplotlib para visualización, ampliando las habilidades aprendidas en la Clase 05.
