# Guía de Uso - Clase 01: Conceptos Básicos de Python

Esta carpeta contiene los primeros programas desarrollados para comprender la sintaxis básica, el manejo de variables y operaciones matemáticas en Python.

## Contenido de la Carpeta

A continuación se detalla el funcionamiento de cada uno de los scripts incluidos:

### 1. Calculadora Básica (`Calculadora.py`)
Este programa implementa una calculadora funcional con un menú interactivo.
*   **Funcionamiento**: Utiliza la librería `math` para realizar operaciones.
*   **Operaciones disponibles**: 
    1. Suma
    2. Resta
    3. Multiplicación
    4. División (con validación para no dividir entre cero)
    5. Exponente
    6. Cuadrado
*   **Cómo usar**: Ejecuta el archivo y sigue las instrucciones en pantalla para elegir una operación e ingresar los números.

### 2. Calculadora de IMC (`IMC.py`)
Un script sencillo para calcular el Índice de Masa Corporal.
*   **Funcionamiento**: Solicita al usuario su peso (kg) y altura (m), aplica la fórmula `peso / (altura^2)` y muestra el resultado.
*   **Cómo usar**: Ejecuta el archivo e ingresa los datos solicitados cuando se te pida.

### 3. Suma de Dos Variables (`Sumas2V.py`)
El programa más básico para entender la entrada de datos y tipos de variables.
*   **Funcionamiento**: Pide dos números (uno entero y uno flotante), realiza la suma y muestra el total.
*   **Cómo usar**: Ejecuta el archivo e ingresa los dos valores numéricos.

## Cómo Ejecutar los Códigos
1. Asegúrate de tener Python instalado.
2. Abre una terminal en esta carpeta.
3. Ejecuta cualquier script usando el comando:
   ```bash
   python nombre_del_archivo.py
   ```
